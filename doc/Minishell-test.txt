-----------------
 ./src/
-> affiche

-----------------
./a.out (permision 000)
-> affiche

Pipe
-----------------
cat | cat | ls
-> affiche ls, attend input pour le premier cat, attend input pour le deuxieme cat

-----------------
exit | exit
-> ne quite pas minishell

-----------------
echo hello | cd 
-> ne change pas de directory

-----------------
cd $NONE
-> change le directory a HOME

-----------------
cd -
-> affiche OLDPWD et change le directory to OLDPWD

Redirection
-----------------
cat < infile | cat > outfile
-> deplace le contenu de infile dans outfile

-----------------
cat < infile1 < infile2 | cat > outfile1 > outfile2
-> deplace le contenu de infile2 dans outfile2 et créer le dossier outfile1

-----------------
cat | ls && cat | cat | ls
-> affiche le premier ls, attend input pour le premier cat, affiche le deuxieme ls, attend input pour le deuxieme cat, attend input pour le troisieme cat



Wildcard
-----------------
echo *
-> affiche tout le contenu du dossier

-----------------
echo */
-> affiche uniquement les dossiers suivit d'un /

-----------------
echo M*
-> affiche 
Makefile

-----------------
echo *e*
-> affiche toute se qui contient un e

-----------------
echo *.c | tr ' ' '\n' | wc -l
-> doit afficher le nombre de fichier .c dans le repertoire

-----------------
cat M*
-> affiche le contenu du Makefile
